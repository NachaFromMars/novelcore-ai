# novelcore-ai — AI novel writing where the author keeps the soul

> A disciplined AI novel writing system: AI is the second pen, the author keeps voice and vision. Precise token budgets, 4-block prompts, and a 6-phase chapter workflow keep prose clean and human.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
NovelCoreAI V2.0 enforces that AI serves the author — not the other way around. Every chapter follows a 6-phase workflow, every beat is written with a 4-block prompt structure, and strict token budgets (400–600 Vietnamese words/beat, 800–1500 tokens/write, 1 beat per prompt always) keep output focused. Anti-score-inflation rules ensure 7-angle reviews stay honest. Context partitioning manages long manuscripts efficiently.

## Features
**4-block prompt structure (every beat):**
1. Context — chapter/beat/situation
2. Task — what the beat must achieve
3. Emotion/Voice — tone, atmosphere, voice name
4. Technical constraints — 400-600 words, 0% markdown, 0% em dash, 0% English, 0% AI words, pure prose

**6-phase chapter workflow:**
Chapter Architecture → Beat Outline → Write (3 versions, pick/mix) → Review (7 angles) → Clean (Markdown+AI) → Save

**Quality controls:** anti-score-inflation, token budget, context partitioning

## Usage / Quick Start
Follow 6-phase workflow. Write one beat per prompt using the 4-block structure. Never combine 2+ beats in one prompt.
